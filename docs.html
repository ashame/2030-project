Site Overview

  • Welcome to Finder, your final destination on finding new places to eat.
  • To start out, you'll need to register for an account from the homepage. This allows you access to our ever-growing search form, with new features to be added in the future.
  • Once you've made an account, you can enter a query (think pizza, sushi, etc) and a location (this can be a city, address, or something specific).
  • You'll shortly see a stack of cards appear on-screen. Click yes to view more about the restaurant, and no to dismiss it and move on to the next.
  • Once you've gone through this stack, you'll be given an option to start a new search.

Features Coming Soon
